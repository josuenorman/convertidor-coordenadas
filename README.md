# Utilidades SIG Web

Aplicación web sencilla para trabajar con coordenadas geográficas y UTM desde el navegador.

Permite ingresar puntos manualmente, convertir coordenadas, visualizarlas en un mapa, organizarlas por proyectos y exportarlas para usarlas en Excel o QGIS.

## ¿Para qué sirve?

Sirve para apoyar trabajos básicos de SIG, levantamiento de información territorial, diagnósticos comunitarios, gestión del riesgo y organización de puntos de campo.

## Funciones principales

- Crear y manejar proyectos de coordenadas.
- Convertir coordenadas WGS84 Geográficas ↔ UTM.
- Agregar puntos manualmente.
- Capturar puntos desde el mapa.
- Visualizar puntos sobre distintas capas base.
- Buscar lugares en el mapa.
- Importar datos desde CSV o Excel.
- Cargar Shapefile en formato ZIP.
- Editar la tabla de coordenadas.
- Exportar datos en CSV y GeoJSON para Excel y QGIS.

## Privacidad

La aplicación no usa servidor ni base de datos. Los proyectos y puntos se guardan localmente en el navegador de cada usuario.
